# Queue Fibonacci App

## 🚀 Technologies
- [Expo](https://expo.dev)
- [React Native](https://reactnative.dev)

## ❓ How to use

### How to install
In the `/src/services/api` file, change the `baseURL` variable to the machine's IP.

```bash
yarn install
expo start
```
